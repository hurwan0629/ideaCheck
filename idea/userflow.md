[무료판 서비스]

1. 서비스 주제 입력
   → "생각해둔 서비스 주제가 있나요?"

2. 결정 분기
   ├─ Yes:
   │   → 서비스 작성 템플릿 제공 (간단 버전)
   │   → 작성된 템플릿 분석
   │   → 관련 경쟁사 / 대체제 / 특징 제공
   │   → End
   │
   └─ No:
       → 가벼운 시장조사 정보 제공

[유료판 서비스]

1. 서비스 주제 입력
   → "생각해둔 서비스 주제가 있나요?"

2. 결정 분기
   ├─ Yes:
   │   → 사업계획 입력
   │      (수익모델, 타겟, 비즈니스 모델, 서비스 주제 등)
   │   → 작성된 내용 기반 분석
   │   → 관련 경쟁사 / 대체제 / 특징 제공
   │   → 구체적인 출처 제공
   │   → 문제점 파악 및 개선 방향 제시
   │   → End
   │
   └─ No:
       → 크롤링 / API / AI 기반 시장분석 정보 제공


[요약]
- 무료:
  → 간단한 아이디어 검증 + 경쟁사 정보 제공

- 유료:
  → 심층 분석 + 출처 기반 데이터 + 개선 방향 제시

- 차별 포인트:
  → 데이터 기반 분석 (RAG, 외부 DB 활용)
  → 비즈니스 프레임워크 적용 (린 스타트업, BM 캔버스)
