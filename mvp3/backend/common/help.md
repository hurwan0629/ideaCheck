이 프로젝트는 scheduler/, app/ 등의 프로젝트에서 쓰일 각종 자원이 들어갈 공간입니다.

core/ - 핵심 api키, db, exception등과 같은 공통적으로 쓰이는 서버 단위의 함수 등이 들어있는 폴더이빈다.

services/ - 비즈니스 로직이 들어있는 폴더입니다. 각종 작업 (크롤링, 분석, crud 등)이 들어있습니다.

repositories/ - DB 조작이 필요할 때 사용 가능한 함수가 들어있는 프로젝트입니다.

models/ 와 schema/ - DB구조와 입출력 데이터 형태를 보여줍ㄴ디ㅏ.

utils/ 특정 반복되는 작업등을 한번에 관리하기 위해서 사용되는 폴더입니다.
