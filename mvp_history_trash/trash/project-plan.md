# 프로젝트 기획안: ValidAI — AI 창업 아이디어 검증 플랫폼

> 작성일: 2026-04-07 | 버전: v1.0

---

## 1. 서비스 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | ValidAI |
| 한 줄 정의 | 창업 아이디어를 입력하면 AI가 시장·경쟁사·수익모델을 즉시 분석해주는 검증 플랫폼 |
| 핵심 고객 | 예비 창업자, 사이드 프로젝트 개발자, 초기 스타트업 팀 |
| 문제 정의 | 창업 아이디어 검증에 평균 수개월·수백만 원이 소요됨. 대부분 혼자 구글링으로 해결하지만 체계적 분석이 어렵고, 컨설팅은 비쌈 |
| 해결 방식 | RAG + AI로 경쟁사·시장 데이터를 즉시 조합해 린 스타트업 프레임워크 기반 보고서 자동 생성 |

---

## 2. 핵심 기능

### 2-1. 무료 플랜
- 서비스 주제 입력 → AI 아이디어 분석 템플릿 자동 완성
- 유사 경쟁사 3~5개 요약 (이름, 특징, 링크)
- 아이디어 강점/약점 간단 리포트 (1페이지)

### 2-2. 유료 플랜 (월 구독)
- 사업계획 전체 입력 (타겟, 수익모델, BM 등)
- 경쟁사 심층 분석 (출처 URL 포함, 최신 데이터 기반)
- 시장 규모 추정 (TAM/SAM/SOM)
- 문제점 파악 + 개선 방향 제시 (린 스타트업 / BM 캔버스 형식)
- PDF 보고서 다운로드

### 2-3. 차별화 포인트
- **출처 기반 분석**: RAG로 실제 뉴스·DB 데이터 인용 (환각 최소화)
- **프레임워크 적용**: 린 스타트업, BM 캔버스 자동 생성
- **빠름**: 입력 후 3분 이내 초안 완성

---

## 3. 비즈니스 모델

| 플랜 | 가격 | 주요 제한 |
|------|------|-----------|
| 무료 | 0원 | 월 3회, 기본 분석 |
| 스타터 | 9,900원/월 | 월 20회, 심층 분석 |
| 프로 | 29,900원/월 | 무제한, PDF, API 연동 |
| 팀 | 별도 문의 | 팀원 공유, 화이트라벨 |

**수익 구조 (12개월 목표)**
- 유료 전환율 목표: 무료 사용자의 5~8%
- 월 1,000명 유료 사용자 시 예상 MRR: ~1,500만 원
- CAC 절감: 콘텐츠 마케팅(창업 블로그, SEO) 중심

## 4. 기술 스택

| 레이어 | 기술 |
|--------|------|
| 프론트엔드 | Next.js 14 (App Router), Tailwind CSS |
| 백엔드 | FastAPI (Python) |
| AI / LLM | GPT API (GPT-4o-mini) |
| RAG | LlamaIndex + Pinecone (벡터 DB) |
| 외부 데이터 | Crunchbase API, Google Trends, 뉴스 크롤러 |
| 인증 | Supabase Auth |
| DB | PostgreSQL (Supabase) |
| 결제 | Stripe |
| 인프라 | Vercel (FE) + Railway (BE) |

**아키텍처 흐름**
```
사용자 입력
  → FastAPI 백엔드
    → RAG 파이프라인 (Pinecone 검색 + 외부 API 조회)
    → GPT API로 구조화된 보고서 생성
  → Next.js 프론트에서 스트리밍 렌더링
```

## 5. 개발 로드맵

### Phase 1 — MVP (0~2개월)
- [ ] 아이디어 입력 → 경쟁사 분석 기본 흐름 구현
- [ ] GPT API 연동 + 프롬프트 설계
- [ ] 무료 플랜 완성
- [ ] 랜딩 페이지 + 베타 대기자 등록

### Phase 2 — 유료 출시 (3~4개월)
- [ ] GPT API를 Claude API로 변환
- [ ] RAG 파이프라인 구축 (Pinecone + 뉴스 크롤러)
- [ ] Stripe 결제 연동
- [ ] PDF 보고서 출력 기능
- [ ] BM 캔버스 자동 생성

### Phase 3 — 성장 (5~8개월)
- [ ] API 제공 (외부 서비스 연동용)
- [ ] 팀 플랜 / 화이트라벨
- [ ] 사용자 대시보드 (히스토리, 비교 분석)
- [ ] 영어 지원 → 글로벌 확장

## 6. 시장 분석

- **국내 예비창업자**: 연 약 15만 명 (중소벤처기업부 2024)
- **유사 해외 서비스**: Validately, IdeaBuddy, Lean Canvas App — 모두 영문, 한국 로컬 데이터 부재
- **경쟁 우위**: 한국어 특화 + 최신 국내 스타트업 생태계 데이터 + 출처 제공

## 7. 리스크 및 대응

| 리스크 | 대응 방안 |
|--------|-----------|
| LLM 환각으로 잘못된 데이터 제공 | RAG + 출처 명시 의무화, 면책 문구 |
| 경쟁사 API 접근 제한 | 공개 데이터(뉴스, 공시) 크롤링 병행 |
| 낮은 유료 전환율 | 무료 제한을 점진적으로 강화, 가치 증명 후 업셀 |
| Claude API 비용 증가 | 프롬프트 캐싱 + 요약 단계 분리로 토큰 절약 |
