성된 구조 요약:

  mvp/
  ├── frontend/  (Next.js 16, 빌드 ✓)
  │   ├── app/          9개 페이지 (/, /analyze, /report/[id], /dashboard, /pricing, /login, /signup, /settings, /trends)
  │   ├── components/   wizard/ + report/ 7개 컴포넌트
  │   ├── lib/          api.ts, stream.ts, auth.ts, format.ts
  │   └── store/        wizard.ts (Zustand)
  │
  └── backend/  (FastAPI)
      └── app/
          ├── routers/  analyze, report, user, trends, webhook (8개 API)
          ├── services/ ai.py (OpenAI gpt-4o-mini), search.py (Tavily), report.py
          ├── models/   report.py, user.py
          └── db/       supabase.py

  시작하려면:
  # 백엔드
  cd mvp/backend && pip install -r requirements.txt
  cp .env.example .env  # API 키 입력
  uvicorn app.main:app --reload

  # 프론트엔드
  cd mvp/frontend
  cp .env.local.example .env.local  # Supabase URL/Key 입력
  npm run dev
