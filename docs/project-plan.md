# 스타트업 아이디어 검증 플랫폼 — 기획안

> 작성일: 2026-04-07  
> 상태: MVP 기획 단계

---

## 1. 서비스 개요

**서비스명 (가칭):** IdeaCheck (또는 런치스코프 / 아이디어레이더)

**한 줄 정의:**  
바이브코딩 시대의 창업자를 위한 AI 기반 스타트업 아이디어 검증 & 시장조사 자동화 서비스

**핵심 가설:**  
Claude Code 등 AI 코딩 도구 덕분에 개발 속도는 빨라졌지만, "무엇을 만들지"를 결정하는 기획·검증 단계는 여전히 수작업에 의존한다. 이 병목을 자동화하면 수요가 있다.

---

## 2. 문제 & 타겟

### 타겟 유저
| 분류 | 설명 |
|------|------|
| 1차 타겟 | 사이드 프로젝트 / 스타트업을 준비하는 1~3인 개발자·기획자 |
| 2차 타겟 | 아이디어는 있지만 시장 검증 경험이 없는 비개발자 창업 준비생 |

### 해결하려는 문제
- 시장조사에 수일이 걸림 (경쟁사 탐색, TAM 추정, 사용자 페인포인트 정리)
- MBA·컨설팅 프레임워크(SWOT, BM 캔버스 등)를 혼자 적용하기 어려움
- "이 아이디어 될까?" 를 빠르게 검증할 도구가 파편화되어 있음

## 3. 서비스 구조

### 3-1. 무료 플랜
```
아이디어 입력
  ├─ 주제 있음 → 간단 템플릿 작성 → AI 분석
  │               → 경쟁사 3~5개 + 대체제 + 차별점 요약
  └─ 주제 없음 → 트렌드 키워드 기반 시장 스냅샷 제공
```
- 리포트 1회 무료, 이후 월 2회 제한
- 워터마크 포함 PDF 다운로드

### 3-2. 유료 플랜 (월 $19 / $49)
```
사업계획 입력
  (수익모델 · 타겟 · BM · 서비스 주제)
  → 심층 분석 리포트
     - 경쟁사 매트릭스 (출처 링크 포함)
     - TAM/SAM/SOM 추정
     - 린 스타트업 가설 캔버스
     - 문제점 + 개선 방향
     - 다음 액션 플랜 (개발 아키텍처 추천 포함)
```

| 항목 | Lite ($19) | Pro ($49) |
|------|-----------|-----------|
| 월 리포트 | 10회 | 무제한 |
| 출처 데이터 | O | O |
| 아키텍처 추천 | X | O |
| API 접근 | X | O |

## 4. 핵심 차별화 — VenturusAI 대비

VenturusAI는 범용 분석 도구다. 우리는 **"빠르게 만들려는 사람"** 에 집중한다.

| 비교 항목 | VenturusAI | 우리 서비스 |
|-----------|-----------|------------|
| 분석 프레임워크 | SWOT·PESTEL 자동화 | 린 스타트업 + BM 캔버스 중심 |
| 출력물 | 리포트 | 리포트 + 다음 액션 플랜 |
| 개발 연계 | 없음 | Claude Code 아키텍처 추천 포함 |
| 데이터 소스 | LLM 내부 | RAG + VC 블로그/뉴스 크롤링 |
| 가격 경쟁 | 하지 않음 | 하지 않음 (포지셔닝 차별화) |

**차별화 핵심:**  
단순 분석을 넘어 **"이 아이디어로 지금 당장 뭘 해야 하는가"** 를 알려주는 액션 중심 리포트

## 5. 기술 스택 (MVP 기준)

| 레이어 | 선택 | 이유 |
|--------|------|------|
| 프론트엔드 | Next.js 14 (App Router) | SSR + 빠른 프로토타이핑 |
| 백엔드 | FastAPI (Python) | AI 파이프라인 연동 용이 |
| AI 엔진 | Claude API (claude-sonnet-4-6) | 긴 컨텍스트 + 구조화 출력 |
| RAG / 검색 | Tavily API + pgvector | 실시간 웹 검색 + 벡터 저장 |
| DB | Supabase (PostgreSQL) | 인증 + DB 통합, 빠른 셋업 |
| 결제 | Stripe | 구독 관리 표준 |
| 배포 | Vercel (FE) + Railway (BE) | 저비용 MVP 운영 |

### AI 분석 파이프라인
```
유저 입력
  → 웹 검색 (Tavily) → 경쟁사·시장 데이터 수집
  → RAG 컨텍스트 구성
  → Claude API → 구조화 분석 (JSON 출력)
  → 리포트 렌더링 (MDX or HTML)
```

## 6. 수익 모델

- **Freemium SaaS**: 무료로 유입 → 유료 전환
- **API 플랜**: Pro 유저에게 분석 API 제공 (빌더·에이전트 연동용)
- **B2B 가능성**: 엑셀러레이터·대학 창업센터에 팀 플랜 판매 (장기 목표)

**손익분기 추정 (보수적):**
- 서버 고정비: 월 ~$150 (Railway + Vercel + API 비용)
- BEP: Lite 플랜 8명 또는 Pro 플랜 4명

## 7. MVP 로드맵

### Phase 1 — MVP (6주)
- [ ] 아이디어 입력 폼 + 무료 리포트 생성
- [ ] Claude API 연동 (경쟁사 분석, SWOT 자동화)
- [ ] Tavily 웹 검색 연동
- [ ] Supabase 인증 + 리포트 저장
- [ ] 랜딩 페이지 + 대기자 이메일 수집

### Phase 2 — 유료화 (4주)
- [ ] Stripe 구독 연동
- [ ] 유료 리포트 (BM 캔버스, TAM 추정, 액션 플랜)
- [ ] PDF 다운로드
- [ ] 사용자 대시보드 (리포트 히스토리)

### Phase 3 — 성장 (8주~)
- [ ] VC 블로그 / Startup Recipe 크롤링 RAG 강화
- [ ] Claude Code 아키텍처 추천 모듈
- [ ] API 플랜 오픈
- [ ] SEO 콘텐츠 (아이디어 카테고리별 시장 리포트 퍼블리싱)
