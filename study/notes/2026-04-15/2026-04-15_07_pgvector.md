# 07. pgvector 실제 사용

## pgvector란

PostgreSQL 확장. 벡터(`list[float]`)를 컬럼으로 저장하고, 유사도 기반 검색을 쿼리로 할 수 있게 해줌.

RAG 흐름에서:
```
텍스트 → 임베딩 API → list[float] → pgvector 컬럼에 저장
쿼리 텍스트 → 임베딩 → DB에서 가장 가까운 벡터들 조회 (Top-K)
```

## SQLAlchemy 모델에서 벡터 컬럼 선언

```python
# app/models/competitor_analysis.py
from sqlalchemy import Column, Integer, Text, DateTime
from sqlalchemy.orm import declarative_base
from pgvector.sqlalchemy import Vector
from datetime import datetime

Base = declarative_base()

class CompetitorAnalysis(Base):
    __tablename__ = "competitor_analyses"

    id = Column(Integer, primary_key=True)
    competitor_id = Column(Integer, nullable=False)
    summary = Column(Text)
    strengths = Column(Text)
    weaknesses = Column(Text)
    embedding = Column(Vector(1536))   # OpenAI text-embedding-3-small 차원수
    created_at = Column(DateTime, default=datetime.now)
```

`Vector(1536)` — 차원 수는 임베딩 모델에 따라 다름:
- OpenAI `text-embedding-3-small`: 1536
- OpenAI `text-embedding-3-large`: 3072
- OpenAI `text-embedding-ada-002`: 1536

## 임베딩 생성 — OpenAI API

```python
from openai import OpenAI

openai_client = OpenAI()  # OPENAI_API_KEY 환경변수에서 자동 읽음

def create_embedding(text: str) -> list[float]:
    response = openai_client.embeddings.create(
        model="text-embedding-3-small",
        input=text,
    )
    return response.data[0].embedding   # list[float], 길이 1536
```

## 임베딩 저장

```python
from sqlalchemy.orm import Session
from app.models.competitor_analysis import CompetitorAnalysis

def save_analysis_with_embedding(db: Session, competitor_id: int, analysis: dict):
    # 임베딩할 텍스트 구성 (검색에 쓸 내용 조합)
    text_for_embedding = f"""
    {analysis['summary']}
    강점: {', '.join(analysis['strengths'])}
    약점: {', '.join(analysis['weaknesses'])}
    특성: {analysis.get('characteristics', '')}
    """.strip()

    embedding = create_embedding(text_for_embedding)

    record = CompetitorAnalysis(
        competitor_id=competitor_id,
        summary=analysis["summary"],
        strengths="\n".join(analysis["strengths"]),
        weaknesses="\n".join(analysis["weaknesses"]),
        embedding=embedding,   # list[float] 그대로 넣으면 됨
    )
    db.add(record)
    db.commit()
```

## 유사도 검색 쿼리

```python
from pgvector.sqlalchemy import Vector
from sqlalchemy import func

def find_similar_competitors(db: Session, query_text: str, top_k: int = 5):
    # 쿼리 텍스트 임베딩
    query_embedding = create_embedding(query_text)

    # cosine distance로 유사도 검색 (<=> 연산자)
    results = (
        db.query(CompetitorAnalysis)
        .order_by(CompetitorAnalysis.embedding.cosine_distance(query_embedding))
        .limit(top_k)
        .all()
    )
    return results
```

**거리 연산자 종류:**
| 연산자 | 의미 | 메서드 |
|---|---|---|
| `<->` | L2 (유클리드) 거리 | `.l2_distance()` |
| `<=>` | Cosine distance | `.cosine_distance()` |
| `<#>` | Inner product (음수) | `.max_inner_product()` |

텍스트 유사도엔 **cosine distance** 가 일반적. 벡터 크기(길이) 영향 없이 방향만 비교.

## RAG 흐름에서 실제 사용 예시

```python
# 사용자가 "구독형 식품 배송 서비스" 입력했을 때

def analyze_idea(db: Session, core_idea: str) -> dict:
    # 1. 유사한 경쟁사 Top-5 조회
    similar = find_similar_competitors(db, core_idea, top_k=5)

    # 2. 경쟁사 데이터를 컨텍스트로 구성
    competitor_context = "\n\n".join([
        f"경쟁사: {c.competitor_id}\n요약: {c.summary}\n강점: {c.strengths}"
        for c in similar
    ])

    # 3. Claude에게 분석 요청
    import anthropic
    client = anthropic.Anthropic()

    response = client.messages.create(
        model="claude-sonnet-4-6",
        max_tokens=2000,
        messages=[{
            "role": "user",
            "content": (
                f"아이디어: {core_idea}\n\n"
                f"유사 경쟁사 데이터:\n{competitor_context}\n\n"
                f"SWOT 분석과 진입 가능성 점수 5개를 JSON으로 제공해줘."
            )
        }]
    )
    return response.content[0].text
```

## 정리

```
pgvector 쓰는 흐름:

1. 텍스트 → OpenAI embedding API → list[float]
2. list[float] → Vector 컬럼에 저장 (db.add + commit)
3. 검색: 쿼리 텍스트 → 임베딩 → .cosine_distance() → .limit(5) → Top-K
4. Top-K를 Claude 프롬프트 컨텍스트에 넣음
```

| 작업 | 코드 |
|---|---|
| 컬럼 선언 | `Column(Vector(1536))` |
| 임베딩 생성 | `openai_client.embeddings.create(...)` |
| 저장 | `record.embedding = embedding` + commit |
| 유사도 검색 | `.cosine_distance(query_embedding)` + `.limit(k)` |
